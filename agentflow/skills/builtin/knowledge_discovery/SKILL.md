# Knowledge Discovery Skill

This skill unifies searching corporate knowledge bases and performing data analysis.

## Features
1. **Hybrid Analysis**: Automatically switches between Vector Search and SQL based on intent.
2. **Auto-Charting**: Generates visualization hints for UI.
3. **Proactive Next Steps**: Suggests actionable next steps via UI2Agent protocol.
