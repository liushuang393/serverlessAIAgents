# コンプライアンス知識ベース

このディレクトリには法規制・コンプライアンス情報を格納します。

## 対象 Agent
- QiAgent（器）: 規制対応の技術要件参照
- ReviewAgent（検証）: コンプライアンスチェック（将来拡張）

## ファイル形式
- `.md` または `.txt` ファイル
- 1ファイル = 1規制/地域推奨

## 内容カテゴリ
- GDPR（EU）
- 中国サイバーセキュリティ法
- 個人情報保護法（日本）
- CLOUD Act（米国）
- 業界固有規制

## 使用例
技術実装時に必要な法規制要件を検索し、コンプライアンス対応を支援します。
