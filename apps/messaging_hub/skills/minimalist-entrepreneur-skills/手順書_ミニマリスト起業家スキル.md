## 3. インストール手順

### 方法A: Claude Codeプラグインとしてインストール（推奨）

Claude Codeのターミナルで以下を実行します：

```bash
# マーケットプレイスを追加
/plugin marketplace add slavingia/skills

# プラグインをインストール
/plugin install minimalist-entrepreneur
```

> **注意**: マーケットプレイスに登録されていない場合は、方法Bを使用してください。

### 方法B: 手動インストール（本セッションで実施済み）

本セッションで以下の構成でファイルを作成済みです：

```
AI作業フォルダ/minimalist-entrepreneur-skills/
├── find-community/SKILL.md
├── validate-idea/SKILL.md
├── mvp/SKILL.md
├── processize/SKILL.md
├── first-customers/SKILL.md
├── pricing/SKILL.md
├── marketing-plan/SKILL.md
├── grow-sustainably/SKILL.md
├── company-values/SKILL.md
└── minimalist-review/SKILL.md
```

Claude Codeで手動登録する場合は、このフォルダを以下のパスにコピーしてください：

```bash
# プロジェクトのスキルフォルダにコピー
cp -r minimalist-entrepreneur-skills/* ~/.claude/skills/
```

または、プロジェクト固有のスキルとして使用する場合：

```bash
# プロジェクトルートに配置
cp -r minimalist-entrepreneur-skills/* <プロジェクトルート>/.claude/skills/
```

## 4. 使い方

### 基本的な使い方

Claude Codeの会話内で、スラッシュコマンドとして呼び出します：

```
/find-community
→ 自分が所属するコミュニティを分析し、ビジネス機会を特定

/validate-idea サブスクリプション型の日本語学習アプリ
→ アイデアの検証フレームワークを実行

/pricing 月額制SaaSツール
→ 最適な価格戦略を提案

/minimalist-review 新しいオフィスを借りるべきか
→ ミニマリスト起業家の視点でレビュー
```

### 推奨ワークフロー

起業の各段階に合わせて、以下の順序でスキルを活用してください：

```
Step 1: /find-community    → コミュニティを見つける
    ↓
Step 2: /validate-idea     → アイデアを検証する
    ↓
Step 3: /processize        → 手動プロセスを設計する
    ↓
Step 4: /mvp               → MVPを定義する
    ↓
Step 5: /first-customers   → 最初の顧客を獲得する
    ↓
Step 6: /pricing           → 価格を最適化する
    ↓
Step 7: /marketing-plan    → マーケティング戦略を立てる
    ↓
Step 8: /grow-sustainably  → 持続可能に成長する
    ↓
Step 9: /company-values    → 企業文化を定義する（採用前）
    ↓
随時:   /minimalist-review → 重要な判断をレビューする
```
